Tailwind CSS v4 is the latest version featuring the new @theme directive, improved performance, and enhanced CSS layers support for better cascade control.

Use the @theme directive to define custom design tokens like colors, fonts, and spacing. You can also use @custom-variant for advanced functionality.

Dark mode is supported through @custom-variant definitions and works seamlessly with the existing dark: variant utilities for responsive theming.
